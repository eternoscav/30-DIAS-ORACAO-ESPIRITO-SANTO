Segundo Dia - Arrebata-me

Arrebata-me, Espírito Santo, no sopro impetuoso do teu amor. Arrebata-me para horizontes mais amplos, para além dos limites de meu pobre espírito. Arrebata-me, elevando todo o meu ser para as alturas da perfeição divina. Arrebata-me para longe de meus sonhos terrestres para a contemplação e o cumprimento dos planos de Deus.
